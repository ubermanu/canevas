Class SpriteMaterial

Loop into multiple images to give an animation.

  src: 'sprite.png',
  frameStart: 0,
  frameCount: 4,
  frameRate: 1,
  frameSize: [16, 16],
  frameOffset: [0, 0],
  loop: true
})
